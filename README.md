# mktable
Generates a ASCII art table from a text file


## Example
Suppose we have a .csv file with the following syntax:<br>
```
Name;Sex;Age;Height(in);Weight(lbs)
Alex;M;41;74;170
Bert;M;42;68;166
Carl;M;32;70;155
Dave;M;39;72;167
Elly;F;30;66;124
Fran;F;33;66;115
Gwen;F;26;64;121
Hank;M;30;71;158
Ivan;M;53;72;175
Jake;M;32;69;143
Kate;F;47;69;139
Luke;M;34;72;163
Myra;F;23;62;98
Neil;M;36;75;160
Omar;M;38;70;145
Page;F;31;67;135
Quin;M;29;71;176
Ruth;F;28;65;131
```
<br>

## Important Notes
I use the [Rainbow](https://github.com/mdk97/Rainbow) repository to print colored text with the tables, so don't forget to `git submodule update --init`
